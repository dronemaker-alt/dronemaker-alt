# LAB NOTES

This directory is the engineering notebook.

Document:

- experiments
- failures
- measurements
- discoveries
- strange ideas
- hardware bring-up
- firmware testing
- diagrams
- observations
- prototype revisions

---

# Suggested Entry Format

```markdown
# Date

## Objective

## Hardware Used

## Firmware / Software

## Observations

## Problems Encountered

## Measurements

## Ideas

## Next Steps
```

---

# Philosophy

Archive everything.

The random experiment from today may become the key system later.

Even failed tests contain valuable engineering data.
