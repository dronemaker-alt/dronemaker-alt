# Drone Libre Research Archive

This folder is the structured mirror library for outside projects, concepts, and reference designs that may help Drone Libre.

## Directory Map

- `airframes/` — drone frames, VTOL structures, optimized arms, printable frames.
- `radar/` — radar, RF sensing, ranging, and ground-station detection projects.
- `robots/` — quadrupeds, walking drones, leg mechanisms, and robotic locomotion.
- `sensors/` — cameras, optical flow, LiDAR, ToF, thermal, GPS, IMU, and sensor-fusion references.
- `companion_computers/` — ESP32, Raspberry Pi, Linux, AI companion, RID, telemetry, and coordination projects.
- `ground_station/` — controllers, displays, antennas, SDR, command stations, and field cases.
- `software/` — firmware, open-source flight control, autonomy, mapping, scanning, and tooling.
- `materials_testing/` — print settings, filament testing, crash testing, and strength notes.
- `source_notes/` — raw source notes before they are cleaned up into a project mirror.

## Mirror Entry Format

Each mirrored project should include source attribution, summary, relevance to Drone Libre, files mirrored or still needed, license status, build/test notes, and next steps.

## Archive Rule

Do not just hoard links. Capture what is useful, note what is unverified, and turn good ideas into testable Drone Libre tasks.
