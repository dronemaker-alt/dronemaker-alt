# MIRROR REPOSITORIES

Offline mirrors of critical GitHub repositories.

Purpose:

- preserve firmware ecosystems
- retain commit history
- survive repository deletion
- maintain offline access

Mirror clones use:

```bash
git clone --mirror
```
