# REFERENCE REPOSITORIES

Curated repositories archived and studied for long-term engineering reference.

These repositories represent:

- flight control systems
- robotics frameworks
- embedded systems
- telemetry architectures
- avionics concepts
- AI integration
- reverse engineering
- sensor systems

# FLIGHT CONTROL SYSTEMS

## INAV
https://github.com/iNavFlight/inav

Navigation-focused multirotor and fixed-wing flight firmware.

## INAV Configurator
https://github.com/iNavFlight/inav-configurator

Ground configuration interface for INAV.

## ArduPilot
https://github.com/ArduPilot/ardupilot

Large-scale autonomous vehicle firmware ecosystem.

## PX4
https://github.com/PX4/PX4-Autopilot

Professional-grade autonomous flight stack.

## Betaflight
https://github.com/betaflight/betaflight

High-performance multirotor flight firmware.

## Cleanflight
https://github.com/cleanflight/cleanflight

Historical predecessor to modern racing-drone firmware ecosystems.

## Paparazzi UAV
https://github.com/paparazzi/paparazzi

Academic and research-oriented autonomous UAV framework.
