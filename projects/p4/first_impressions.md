# LILYGO T-Display P4 — First Impressions

## Initial Arrival
Board arrived smaller than expected.
Photos online made it appear closer to a handheld console size, but actual dimensions are much more compact and practical for portable systems.

Immediate impression:
- feels more like a field instrumentation device
- compact cyberdeck component
- portable avionics/telemetry node

## Packaging
Arrived in hard foam insert case.
Presentation had strong “prototype hardware recovered from a lab” energy.

## Power-Up Experience
Documentation included:
- approximately none

Initial uncertainty:
- multiple USB ports
- unclear power/programming interface
- no startup instructions

Observed:
- USB port labeled `P4U`
- connected safely
- battery charging indicator active
- no smoke released into atmosphere

Discovered:
- physical side-mounted slide power switch required for startup

## First Reactions
Strong potential for:
- drone telemetry terminal
- robot UI controller
- portable diagnostics system
- Cali Prototype Systems handheld ecosystem
- embedded cyberpunk nonsense
